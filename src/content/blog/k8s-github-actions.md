## 为什么要预留接口扩展

你后面很可能会想加：

- 阅读量统计
- 评论系统
- 文章推荐
- 邮件订阅

如果前台骨架从一开始就支持 Cloudflare Worker 路线，这些能力可以逐步长出来，而不需要推翻站点。
