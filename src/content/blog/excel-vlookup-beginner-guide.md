## 适合谁看

这篇文章适合第一次接触 `VLOOKUP` 的 Excel 新手。

你不需要先理解一堆函数概念，只要跟着示例做一遍，就能掌握最常见的用法：**根据一个编号，去另一张表里查出对应信息**。

本文用两个示例文件来讲：

- `示例1-员工字典表.xlsx`：员工资料表，包含员工编号、姓名、部门、职级等信息。
- `示例2-销售订单表.xlsx`：销售订单表，只有员工编号，需要补充姓名、部门、职级。

## VLOOKUP 是做什么的

先用一句人话理解：

```text
我手里有一个编号，去另一张表里找这个编号，找到以后，把同一行的其他信息拿回来。
```

比如订单表里只有员工编号：

| 员工编号 | 姓名 |
| --- | --- |
| E003 | ？ |

员工字典表里有完整资料：

| 员工编号 | 姓名 | 部门 | 职级 |
| --- | --- | --- | --- |
| E003 | 王五 | 技术部 | P7 |

`VLOOKUP` 要做的事就是：

```text
拿 E003 去员工字典表里找
找到 E003 那一行
把姓名“王五”拿回来
```

## 第一步：查出姓名

先同时打开这两个 Excel 文件：

1. `示例1-员工字典表.xlsx`
2. `示例2-销售订单表.xlsx`

然后进入 `示例2-销售订单表.xlsx` 的 `订单明细` 工作表。

你会看到类似这样的表：

| 订单号 | 员工编号 | 姓名(待查) | 部门(待查) | 职级(待查) |
| --- | --- | --- | --- | --- |
| SO001 | E003 |  |  |  |

现在要在 `C2` 单元格查出姓名。

点击 `C2`，输入公式：

```text
=VLOOKUP(B2,'[示例1-员工字典表.xlsx]员工信息'!$A$2:$G$11,2,FALSE)
```

按回车后，如果 `B2` 是 `E003`，`C2` 会显示：

```text
王五
```

这说明 Excel 已经用 `E003` 去员工字典表里查到了姓名。

## 公式拆开看

刚才的公式是：

```text
=VLOOKUP(B2,'[示例1-员工字典表.xlsx]员工信息'!$A$2:$G$11,2,FALSE)
```

不用怕，它本质上只有 4 个参数：

```text
=VLOOKUP(要找谁, 去哪里找, 返回第几列, 是否精确匹配)
```

对应到这条公式：

| 公式部分 | 含义 | 解释 |
| --- | --- | --- |
| `B2` | 要找谁 | 要找 `B2` 里的员工编号，也就是 `E003` |
| `'[示例1-员工字典表.xlsx]员工信息'!$A$2:$G$11` | 去哪里找 | 去员工字典表的 `A2:G11` 范围里找 |
| `2` | 返回第几列 | 找到后返回第 2 列，也就是姓名 |
| `FALSE` | 精确匹配 | 必须一模一样才算找到 |

最重要的是记住这句话：

```text
VLOOKUP = 拿一个值，去查找区域的第一列找，找到后返回右边某一列。
```

## 为什么一定要写 FALSE

新手先记住一个规则：

```text
查员工编号、订单号、学号、手机号这类固定编号时，最后一项一律写 FALSE。
```

也就是：

```text
=VLOOKUP(B2,查找区域,返回第几列,FALSE)
```

`FALSE` 表示精确匹配。只有编号完全一样，Excel 才认为找到了。

如果省略最后一个参数，或者写成 `TRUE`，Excel 可能会按近似匹配去找。结果有时看起来能用，但实际可能已经查错了。

## 向下填充其他行

`C2` 查出姓名后，把鼠标放到 `C2` 右下角的小方块上。

当鼠标变成黑色小十字时，可以双击，也可以向下拖动到最后一行。

这样其他订单也会自动查出姓名。

如果某一行的员工编号是 `E999`，而员工字典表里没有这个编号，Excel 会显示：

```text
#N/A
```

`#N/A` 的意思就是没找到。

## 用 IFERROR 显示友好提示

如果不想让用户看到 `#N/A`，可以用 `IFERROR` 包一层。

把 `C2` 的公式改成：

```text
=IFERROR(VLOOKUP(B2,'[示例1-员工字典表.xlsx]员工信息'!$A$2:$G$11,2,FALSE),"未知员工")
```

它的意思是：

```text
如果 VLOOKUP 查到了，就显示姓名。
如果没查到，就显示“未知员工”。
```

之后再把公式向下填充。

## 查部门和职级

前面查姓名时，公式里的第 3 个参数写的是 `2`：

```text
=IFERROR(VLOOKUP(B2,'[示例1-员工字典表.xlsx]员工信息'!$A$2:$G$11,2,FALSE),"未知员工")
```

这里的 `2` 表示返回员工字典表查找区域里的第 2 列。

假设员工字典表的列顺序是：

| 第几列 | 字段 |
| --- | --- |
| 第 1 列 | 员工编号 |
| 第 2 列 | 姓名 |
| 第 3 列 | 部门 |
| 第 4 列 | 职级 |
| 第 5 列 | 月薪 |
| 第 6 列 | 入职日期 |
| 第 7 列 | 城市 |

那么：

- 查姓名：返回第 `2` 列
- 查部门：返回第 `3` 列
- 查职级：返回第 `4` 列

### D2 查部门

点击 `D2`，输入：

```text
=IFERROR(VLOOKUP(B2,'[示例1-员工字典表.xlsx]员工信息'!$A$2:$G$11,3,FALSE),"未知部门")
```

然后向下填充。

### E2 查职级

点击 `E2`，输入：

```text
=IFERROR(VLOOKUP(B2,'[示例1-员工字典表.xlsx]员工信息'!$A$2:$G$11,4,FALSE),"未知职级")
```

然后向下填充。

到这里，姓名、部门、职级就都能根据员工编号自动查出来了。

## 查找区域里的 $ 是什么

公式里有一段：

```text
$A$2:$G$11
```

这里的 `$` 叫锁定引用。

它的作用是：

```text
向下填充公式时，员工字典表的查找范围不要跟着变化。
```

如果不加 `$`，你向下拖公式时，查找区域可能从：

```text
A2:G11
```

变成：

```text
A3:G12
```

这样查找范围就偏了，结果也可能出错。

所以新手先记一个简单规则：

```text
VLOOKUP 的查找区域要用 $ 锁住。
```

推荐写法：

```text
$A$2:$G$11
```

不推荐写法：

```text
A2:G11
```

## VLOOKUP 的 4 个关键规则

### 规则 1：查找值必须在查找区域第一列

`VLOOKUP` 只能在查找区域的第一列找。

比如查找区域是：

```text
$A$2:$G$11
```

那么 Excel 会在 `A` 列找员工编号。

如果想用员工编号查姓名、部门、职级，这没有问题。  
但如果想用姓名反过来查员工编号，普通 `VLOOKUP` 就不太适合。

### 规则 2：查编号时最后写 FALSE

正确写法：

```text
=VLOOKUP(B2,区域,2,FALSE)
```

不推荐写法：

```text
=VLOOKUP(B2,区域,2)
```

因为省略最后一个参数时，Excel 可能会使用近似匹配。

### 规则 3：查找区域要锁住

正确写法：

```text
$A$2:$G$11
```

不推荐写法：

```text
A2:G11
```

尤其是要向下填充公式时，这一点很重要。

### 规则 4：返回第几列从查找区域开始数

如果查找区域是：

```text
$A$2:$G$11
```

那么 `A` 列是第 1 列，`B` 列是第 2 列，`C` 列是第 3 列。

如果查找区域改成：

```text
$B$2:$G$11
```

那么 `B` 列才是第 1 列。

也就是说，返回第几列不是看 Excel 原始列号，而是看你选中的查找区域。

## 如果两个表在同一个文件里

跨文件公式会比较长。练习时也可以把两个表复制到同一个 Excel 文件里。

假设同一个文件里有这些工作表：

```text
订单明细
员工信息
产品折扣阶梯
```

这时公式可以简化很多。

查姓名：

```text
=IFERROR(VLOOKUP(B2,员工信息!$A$2:$G$11,2,FALSE),"未知员工")
```

查部门：

```text
=IFERROR(VLOOKUP(B2,员工信息!$A$2:$G$11,3,FALSE),"未知部门")
```

查职级：

```text
=IFERROR(VLOOKUP(B2,员工信息!$A$2:$G$11,4,FALSE),"未知职级")
```

这个版本更短，也更适合刚开始练习。

## 近似匹配：根据数量查折扣率

前面讲的是精确匹配。  
`VLOOKUP` 也可以做近似匹配，比如根据订单数量查询折扣率。

假设 `产品折扣阶梯` 工作表里有这样的数据：

| 最低数量 | 折扣率 |
| --- | --- |
| 1 | 100% |
| 10 | 95% |
| 50 | 90% |
| 100 | 85% |
| 500 | 80% |

意思是：

```text
数量达到 1 件，按 100%。
数量达到 10 件，按 95%。
数量达到 50 件，按 90%。
数量达到 100 件，按 85%。
数量达到 500 件，按 80%。
```

比如数量是 `55`，它没有刚好等于 `50` 或 `100`。  
这时 Excel 会找不超过 `55` 的最大档位，也就是 `50`，所以折扣率是 `90%`。

在 `订单明细` 工作表里，点击 `I2`，输入：

```text
=VLOOKUP(G2,'[示例1-员工字典表.xlsx]产品折扣阶梯'!$A$2:$B$6,2,TRUE)
```

这里最后写的是 `TRUE`，表示近似匹配。

近似匹配有一个硬要求：

```text
查找区域第一列必须从小到大排序。
```

也就是最低数量必须按 `1、10、50、100、500` 这样的顺序排列，不能乱序。

折扣率查出来后，`J2` 可以计算折后金额：

```text
=G2*H2*I2
```

含义是：

```text
数量 × 单价 × 折扣率
```

## 常见错误

看到错误不要慌，先按下面这张表排查。

| 看到的错误 | 解释 | 处理办法 |
| --- | --- | --- |
| `#N/A` | 没找到 | 检查编号是否存在，或者用 `IFERROR` 显示友好提示 |
| `#REF!` | 引用坏了 | 检查文件是否改名、移动，或者列号是否超出范围 |
| `#VALUE!` | 公式参数有问题 | 检查逗号、区域、列号是否写错 |
| 查出来不对 | 可能用了近似匹配 | 把最后一个参数改成 `FALSE` |
| 向下拖后出错 | 查找区域没锁住 | 把区域写成 `$A$2:$G$11` |

## 新手检查清单

写完公式后，可以按这个顺序检查：

- 第 1 个参数是不是员工编号单元格，比如 `B2`
- 第 2 个参数是不是员工字典表范围，比如 `$A$2:$G$11`
- 查找区域第一列是不是员工编号
- 查姓名是不是写 `2`
- 查部门是不是写 `3`
- 查职级是不是写 `4`
- 查编号时最后是不是写了 `FALSE`
- 向下填充前，查找区域有没有用 `$` 锁住

## 先练熟这 3 个公式

第一次学 `VLOOKUP`，先不用急着学太多高级写法。

把下面 3 个公式练熟，日常办公里已经能解决很多问题。

查姓名：

```text
=IFERROR(VLOOKUP(B2,'[示例1-员工字典表.xlsx]员工信息'!$A$2:$G$11,2,FALSE),"未知员工")
```

查部门：

```text
=IFERROR(VLOOKUP(B2,'[示例1-员工字典表.xlsx]员工信息'!$A$2:$G$11,3,FALSE),"未知部门")
```

查职级：

```text
=IFERROR(VLOOKUP(B2,'[示例1-员工字典表.xlsx]员工信息'!$A$2:$G$11,4,FALSE),"未知职级")
```

## VLOOKUP 的局限

`VLOOKUP` 很常用，但它也有几个限制：

1. 只能从左往右查。
2. 查找值必须在查找区域第一列。
3. 多条件查找不方便。
4. 跨工作簿时公式会比较长。

如果你使用的是 Excel 365 或 Excel 2021，以后可以继续学习 `XLOOKUP`。

比如根据员工编号查姓名，也可以写成：

```text
=XLOOKUP(B2,'[示例1-员工字典表.xlsx]员工信息'!$A$2:$A$11,'[示例1-员工字典表.xlsx]员工信息'!$B$2:$B$11,"未知员工")
```

不过第一遍学习时，不建议一上来就换函数。  
先把 `VLOOKUP` 的查找逻辑练熟，再学 `XLOOKUP` 会更顺。

## 总结

`VLOOKUP` 可以先记成一句话：

```text
拿一个值，去另一张表第一列找，找到后返回同一行右边某一列。
```

最常用模板是：

```text
=IFERROR(VLOOKUP(要查的值,查找区域,返回第几列,FALSE),"没找到时显示什么")
```

本文最核心的公式是：

```text
=IFERROR(VLOOKUP(B2,'[示例1-员工字典表.xlsx]员工信息'!$A$2:$G$11,2,FALSE),"未知员工")
```

记住这几个点就够了：

- `B2`：要查哪个员工编号
- `$A$2:$G$11`：去员工字典表哪里查
- `2`：返回姓名这一列
- `FALSE`：必须精确匹配
- `IFERROR`：找不到时不要报错，显示“未知员工”
